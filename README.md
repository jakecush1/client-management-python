### How to run:
### Install Python 3.9.19
```
python3 --version
```
### Install PyQt6
```
pip install PyQt6
```

### Clone repo
```
git clone https://github.com/jakecush1/client-management-python.git
```

### Navigate to main directory
```
cd client-management-python/
cd medical_clinic_system/
```
### Run the GUI
```
python3 -m clinic gui
```
